# 🚀 AI Habit Tracker

An AI-powered habit tracking application that helps users build consistency, track daily habits, and receive intelligent insights based on their progress and activity patterns.

The application allows users to create habits, mark them as completed daily, monitor progress through statistics and visual analytics, and receive AI-generated feedback and recommendations using the Gemini Flash 2.5 model.

---

## 🌐 Live Demo
👉 https://habbit-tracker-snowy.vercel.app

---

## ✨ Features

- ✅ Create and manage daily habits
- 📅 Track daily habit completion
- 📊 Interactive statistics dashboard
- 🤖 AI-generated habit insights and analysis
- 🔥 Streak tracking and progress monitoring
- 🌙 Dark/Light theme support
- 📈 Visual charts and analytics
- 🔐 Secure authentication system
- ⚡ Responsive modern UI

---

## 🧠 AI Integration

This project integrates Google's **Gemini Flash 2.5** model to generate:

- Personalized habit insights  
- Productivity analysis  
- Recovery suggestions  
- Weekly performance reports  
- Smart habit improvement recommendations  

The AI analyzes user habit history and generates meaningful feedback that helps users understand their behavior patterns and improve consistency over time.
